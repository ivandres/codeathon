Angular MVC

The view is a compiled angular template like example1.html or example2.html. A template is essentially HTML with additional markup, when it is compiled to a DOM it serves as the view.

The scope is the context where the model is stored. The model is a javaScript object. Two way data binding between the view and the controller keeps this model as the single source of truth. The view is updated automatically whenever the model changes, and the model is updated whenever the view changes.

A controller contains the logic to handle the view and model binding of the $scope it is assigned. Everytime you make a new controller, it get's a new $scope.
