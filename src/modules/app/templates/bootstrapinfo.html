Bootstrap

Bootstrap is a HTML, CSS, and JS framework for developing responsive, mobile first projects on the web.

Bootstrap & JavaScript require JQuery. With Angular, the DOM (Document Object Model) is abstracted away from the developer - JQuery shouldn't be used!

Unless we're talking Angular 2.0, but that's a different story

We use angular-strap instead. Angular strap provides angular directives (see Component section for more details) for bootstrap components. It's what made this modal!
