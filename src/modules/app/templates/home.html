Angular JS is a framework for creating single page web applications. This means that you can swap out html partials without reloading the page using angular routes. Some simple routes have been provided in the nav bar above. Get started with some basic info below and look for some helpful comments in the code!

Data Binding

At the core of the Angular framework is the view, scope and controller which use the Model View Controller design pattern.

Components

Components are small, simple, reusable and self-contained pieces of code that encapsulate a specific behavior. The AngularJS Framework provides services, directives, controllers and filters as a way to easily componentize applications.

Bootstrap

Access to components such as icons, dropdowns, buttons, and navigation and CSS including an awesome responsive grid system! Angular-Strap provides Angular directives for easy integration of these Bootstrap components
